# The Bucket

# Overview

The Bucket is an interactive tool to guide you through traveling the world. 
Click on a country to view relevant information and images, and add your own experiences to a journal.  

# Compatibility

• Chrome
• Firefox
• Microsoft Edge
• Internet Explorer
